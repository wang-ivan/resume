自我介紹

您好，我叫王世宇，希望能夠應徵軟體資訊相關的實習。雖然我要應徵的工作與我上面的文組科系不是很搭，或許您已經開始產生了這份履歷上的這個人並不適任這份工作的刻板印象。不過在您要滑掉之前，希望您能看看下面的一些作品和經歷，期望能讓讓您有一點點的改觀。

在就讀中興大學期間修過57學分（底下有放歷年成績單證明）的資訊管理相關課程，資料結構、物件導向、資料庫等核心課程都有修過，並且製作過多個作品（皆放置於GitHub），所以對於軟體開發領域有一定的經驗，在大學時修習的資訊相關課程中大部分的成績都是班排得中前段，因此我認為我具有與中興大學資管系學生同等的能力。

在網頁前端設計上面，我下過許多功夫，作品集的內容大多都與其有關，而資管系的前端課程只有一堂選修課，所以如果不特別去學的話，就完全不會，因此在這點我能產生出一定的優勢。此外我還熟悉 Linux 的操作，本校資管系是完全沒有 Linux 相關課程的，而我有修過資工系所開的「 UNIX 系統與 Script 程式設計」，因此能夠使用 CLI 操作作業系統，並且會使用現在大部分學生都不會用的 Vim 編輯器。因為本身是文組科系，所以除了具有寫程式的邏輯外，也多了對於「人」更多的關心，因此設計上會更在乎使用者體驗。

期望能有與您共事的機會。
